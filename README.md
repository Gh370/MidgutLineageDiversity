# MidgutLineageDiversity
Important codes and data for "Longitudinal lineage tracing reveals early clonal attrition during *Drosophila* midgut aging".

## code
The calculation process of obtaining effective population size and clonal diversity in midgut.

## data
Files with **annotation** in the filename are the annotations of the phylogenetic trees, which are about the numbers of mutations in the sampled readout sequences. The names of samples are derived from "Longitudinal lineage tracing reveals early clonal attrition during *Drosophila* midgut aging".
### IQ-TREE
Phylogenetic trees constructed using IQ-TREE.
### Phylotime
Time-scaled phylogenetic trees constructed by Phylotime.
 * Phylotime_WholeMidgut.RData: time-scaled phylogenetic trees of cell populations in whole midgut.
 * Phylotime_PosteriorMidgut.RData: time-scaled phylogenetic trees of intestinal stem cell population in posterior midgut.
### Sequence
500 bp readout sequences used to build the phylogenetic trees.
### GraphStructs
Edge lists and node feature of intestinal stem cell lineages.
